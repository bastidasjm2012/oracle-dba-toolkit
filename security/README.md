# Security Toolkit (Oracle 19c)

## Overview

This module contains practical scripts and configuration samples for Oracle Database 19c security administration.

## Key Areas

* Transparent Data Encryption (TDE)
* Keystore and wallet management
* Tablespace encryption
* Oracle Net encryption
* Wallet validation

## Included

* TDE wallet configuration
* Keystore creation and opening
* Master key creation
* Auto-login keystore setup
* USERS tablespace encryption
* Oracle Net encryption sample configuration
* Validation scripts

## Documentation

The following implementation guides are included in the `/docs` folder:

* `enable_tde_oracle_19c.pdf`
* `encrypt_data_files_oracle_19c.pdf`
* `enable_network_encryption_oracle_19c.pdf`

## ACE Contribution

This module demonstrates real-world Oracle security configuration aligned with practical DBA operations and Oracle MAA-style secure environment preparation.
