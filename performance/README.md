# Performance Toolkit (Oracle 19c)

## Overview

Practical scripts for SQL performance analysis and tuning in Oracle Database 19c.

## Key Areas

* SQL Profile management
* SQL execution time analysis from AWR
* Execution plan comparison
* SQL Plan Baseline management
* Long-running SQL monitoring
* ASH-based troubleshooting

## Environment

* Oracle Database 19c
* AWR / ASH licensed environments
* SQL*Plus / SQLcl

## Scripts Included

* `sql_profile_management.sql`
* `execution_time_from_awr.sql`
* `compare_execution_plans.sql`
* `sql_plan_baseline_management.sql`
* `top_sql_by_ash.sql`
* `long_running_sql_check.sql`

* Usage

Run scripts using SQL*Plus:

sqlplus / as sysdba
@sql_profile_management.sql

## ACE Contribution

This module demonstrates real-world SQL performance diagnostics and tuning practices used in Oracle production environments.
