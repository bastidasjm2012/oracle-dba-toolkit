# Data Guard Switchover Runbook

## Objective

Perform a controlled role transition between primary and standby databases.

## Steps

### 1. Validate Configuration

```sql
DGMGRL> show configuration;
DGMGRL> validate database <db_name>;
```

### 2. Perform Switchover

```sql
DGMGRL> switchover to <standby_db>;
```

### 3. Validate Roles

```sql
SELECT database_role FROM v$database;
```

## Notes

* Ensure no apply lag before switchover
* Validate connectivity between nodes
