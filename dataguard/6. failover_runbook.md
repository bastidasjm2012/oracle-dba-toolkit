# Data Guard Failover Runbook

## Objective

Promote standby database to primary in disaster scenarios.

## Steps

### 1. Validate Standby

```sql
DGMGRL> validate database <standby_db>;
```

### 2. Execute Failover

```sql
DGMGRL> failover to <standby_db>;
```

### 3. Reinstate Old Primary

```sql
DGMGRL> reinstate database <old_primary>;
```

## Notes

* Used when primary is unavailable
* May cause data loss depending on protection mode
