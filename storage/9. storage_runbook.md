# Storage Troubleshooting Runbook

## Objective

Diagnose and manage storage issues in Oracle environments.

## Workflow

### 1. Check tablespace usage

```sql
@tablespace_usage.sql
```

### 2. Identify large segments

```sql
@reclaim_space.sql
```

### 3. Analyze growth

```sql
@segment_growth_analysis.sql
```

### 4. Validate ASM

```sql
@asm_diskgroup_usage.sql
@asm_rebalance_monitor.sql
```

### 5. Identify resize opportunities

```sql
@datafile_resize_candidates.sql
```

### 6. Validate corruption

* RMAN
* DBVERIFY
* ANALYZE

## Notes

* Always validate before resizing
* Monitor ASM operations carefully
* Document findings before actions
