# Corruption Detection (Oracle 19c)

## RMAN Validation

rman target /
BACKUP CHECK LOGICAL VALIDATE DATABASE;
```

## DBVERIFY

```bash
dbv file=<datafile> blocksize=8192
```

## ANALYZE

```sql
ANALYZE TABLE schema.table_name VALIDATE STRUCTURE CASCADE;
```

## Notes

* Use RMAN for full database validation
* DBVERIFY works offline or with copies
* ANALYZE locks the object
