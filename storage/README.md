# Storage Toolkit (Oracle 19c)

## Overview

Collection of scripts and procedures for managing storage in Oracle Database 19c environments.

## Key Areas

* Tablespace monitoring
* Space reclamation
* Segment growth analysis
* ASM diskgroup usage
* Rebalance operations
* Datafile optimization
* Corruption detection

## Environment

* Oracle Database 19c
* ASM (Automatic Storage Management)
* Enterprise environments (RAC / Data Guard / OCI)

## Scripts Included

* tablespace_usage.sql
* reclaim_space.sql
* segment_growth_analysis.sql
* asm_diskgroup_usage.sql
* asm_rebalance_monitor.sql
* datafile_resize_candidates.sql

## ACE Contribution

This module demonstrates real-world storage management and troubleshooting scenarios in Oracle environments.
