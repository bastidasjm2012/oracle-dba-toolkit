# Common SRVCTL Commands (Oracle RAC)

## Database

```bash
srvctl status database -d <db>
srvctl start database -d <db>
srvctl stop database -d <db>
```

## Instances

```bash
srvctl status instance -d <db>
srvctl start instance -d <db> -i <inst> -n <node>
srvctl stop instance -d <db> -i <inst> -n <node>
```

## Services

```bash
srvctl status service -db <db>
srvctl config service -db <db>
srvctl add service -db <db> -service <svc> -preferred <inst1> -available <inst2>
```

## SCAN

```bash
srvctl status scan
srvctl status scan_listener
```

## Cluster

```bash
crsctl stat res -t
olsnodes -n
```
