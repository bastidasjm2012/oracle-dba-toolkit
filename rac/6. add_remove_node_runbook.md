# RAC Node Management Runbook

## Objective

Add or remove nodes in an Oracle RAC cluster.

## Add Node (High-Level)

1. Prepare OS (users, groups, packages)
2. Configure SSH equivalence
3. Run Grid Infrastructure add node:

```bash
addNode.sh -silent
```

4. Verify cluster:

```bash
olsnodes -n
crsctl stat res -t
```

---

## Remove Node

1. Stop services on node
2. Remove node from cluster:

```bash
rootcrs.sh -deconfig -force
```

3. Verify cluster health

---

## Notes

* Always validate before/after
* Ensure no active workload on node
